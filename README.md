place `HytaleServer.jar` into `libs` folder.


To test the mod, create in the Server folder a folder named `earlyplugins` and drop Hyxin and this mod compiled.
Hyxin can be found here: https://curseforge.com/hytale/mods/hyxin it adds support for Mixin and MixinExtras.

You can find a guide to set up this project and compile it in: https://www.youtube.com/watch?v=orArE7KSKYs
